# 🎮 PUBG Interactive Showcase

Uma página web interativa inspirada no jogo PUBG: BATTLEGROUNDS, desenvolvida com foco em design moderno, responsividade e interatividade com JavaScript.

---

## 🚀 Sobre o projeto

Este projeto simula uma página real de apresentação de jogo, contendo informações detalhadas, galeria interativa e layout profissional.

O objetivo foi praticar:

* Estruturação semântica com HTML
* Estilização avançada com CSS (Flexbox e Grid)
* Manipulação do DOM com JavaScript
* Criação de interfaces modernas e responsivas

---

## ✨ Funcionalidades

* 🎬 Carrossel de imagens interativo (com navegação por botões e miniaturas)
* 🎮 Seção de apresentação do jogo
* 📊 Informações e estatísticas
* 💻 Requisitos mínimos e recomendados
* 💬 Sistema visual de comentários
* 📱 Layout responsivo para diferentes telas
* 🎨 Design dark moderno com destaque em amarelo

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
